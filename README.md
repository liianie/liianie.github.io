# [好的爱情，是让彼此变得柔软](https://liianie.github.io/)

年轻的时候，我们总以为爱情要轰轰烈烈，要炽热到失去理智。
后来才发现，真正好的爱情，反而是安静的、温柔的。

在一段健康的关系里，男人不需要一直逞强，女人也不必时刻坚强。
男人可以卸下防备，说出自己的疲惫和脆弱；女人也可以放下敏感，不再反复试探对方的真心。

好的爱情，会让一个急躁的人慢下来，让一个冷漠的人学会关心。
你会开始在意对方的感受，而不是只顾自己的情绪；你会愿意为对方改变，却不会迷失自己。

争吵不可避免，但真正相爱的男女，吵完之后会更珍惜。
他们不会用冷战惩罚彼此，也不会用分手当威胁，而是愿意坐下来，把问题说清楚。

爱情最美的状态，不是你离不开我，而是我有你更好，没有你也能好好生活。
当两个人都足够独立，又愿意彼此依靠，这样的感情，才经得起时间的考验。
